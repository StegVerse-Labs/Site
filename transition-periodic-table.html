Experimental Engine

The table is a 2D projection of a 3D transition lattice. The engine selects eligible experiments, writes run manifests, emits knowledge deltas, updates evidence, generates receipts, and queues anomalies for human review.

  • Primitive row: admissibility, conservation, bounded action, and capacity margin.
  • Lag row: observation lag, decision lag, actuation lag, and trust drift.
  • Coupled row: two-state coupling, multi-agent composition, conflict, and consensus.
  • Evidence row: receipt binding and reconstruction.
  • Open-boundary row: irreversibility and guarded self-modification.

Transition Elements as Action Authority

Transition elements are not only research categories. In the StegVerse implementation, each discovered element can unlock a bounded action class: what may be done, where it may act, what receipt is required, and when the action must stop.

This page reads the live transition authority map from data/transition-table/transition-element-action-authority-map-v1.json. When new actions unlock through table discovery, the data file changes and this section updates automatically.

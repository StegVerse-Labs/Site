Audit is evidence. Commit-time admissibility is control.

Logs can explain what happened. A commit boundary determines whether an action is still valid before it becomes an irreversible effect.

The StegVerse pattern is simple: reconstruct current state, re-derive current authority, recompute admissibility, then allow, deny, or fail closed.

The Transition Table is the public map of those boundary decisions. It may describe public posture, but it does not replace the receipt-producing proof authority.

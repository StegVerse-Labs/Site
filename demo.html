What is happening?

1. Commit Boundary

The commit_states array defines the geometric boundary of valid state space at commit time.

2. GOD Divergence

Geometrically Ontological Divergence measures how far the proposed state sits from the boundary. GOD = 1.0 is the surface.

3. Verdict

ALLOW — within boundary. DENY — outside, recoverable. FAIL-CLOSED — catastrophic divergence.

4. Receipt

SHA-256 hash of the canonical record. Anyone with the same inputs can reproduce it.

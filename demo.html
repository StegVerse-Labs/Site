How to use this demo

  • Default Data restores the original sample dataset. It does not evaluate automatically.
  • User Data clears the data window so you can paste or type your own JSON.
  • Random Data fills the window with a random dataset. It does not evaluate automatically.
  • Evaluate is the only button that commits the current data to SCW for Tier 1/Tier 2 evaluation.

Required shape: commit_states must be an array of numeric vectors, and proposed_state must be one numeric vector of the same length.

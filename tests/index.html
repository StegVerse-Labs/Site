Governance
Testing Suite.

These pages let visitors test what a claim, answer, proposed action, or generated output is allowed to become after StegVerse governance is applied.

Run Governance Filter LLM Comparison Transition Admissibility Receipt Replay Fail-Closed
Boundary: this suite does not host an LLM and does not prove that an LLM answer is true. It tests what pasted output is allowed to become under public governance posture.

Testing doctrine

The question is not:
  Is the LLM answer good?

The governing question is:
  What is this answer allowed to become?

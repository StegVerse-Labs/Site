# Site Mirror Proof Boundary

## Rule

```text
Site is a public mirror, not proof authority.
```

## What Site May Do

```text
publish status
explain proof claims
link to artifacts
display task IDs
mirror roadmap state
summarize reports
```

## What Site May Not Do

```text
install files
approve packets
activate node status
authorize FinCo
accredit production
claim canonical authority
override formalism-tests
override master-records
make StegVerse-001 sovereign
```

## Required Public Language

Any public page discussing the completed 31-stage roadmap should include:

```text
Production means accredited participation, not sovereign authority.
```

Any public page discussing Stage 30 packets should include:

```text
The packet is portable evidence of a proposed governed transition, not installation authority.
```

Any public page discussing Stage 28 install plans should include:

```text
An install plan is a candidate transition, not installation authority.
```

Any public page discussing Stage 27 discovery should include:

```text
Discovery observes, models, compares, classifies, and proposes. Discovery does not install.
```

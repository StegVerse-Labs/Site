# StegVerse Evidence and Artifact Map

## Purpose

This map identifies the current evidence supporting the Texas Tech–NVIDIA collaboration concept. It separates verified repository records from asserted or cross-repository artifacts that require independent path and version verification before external outreach.

This document does not claim Texas Tech or NVIDIA affiliation, sponsorship, validation, compute access, or endorsement.

## Research claim under examination

StegVerse is an experimental governance and evidence-continuity framework for agentic AI systems. The proposed research asks whether commit-time authority, valid policy and delegation, transition admissibility, execution-boundary compliance, and post-event reconstruction can remain independently verifiable across distributed multi-agent workloads.

## Evidence maturity scale

```text
E0 — asserted concept only
E1 — documented schema or design
E2 — executable local artifact
E3 — repeatable repository test with expected outcomes
E4 — current-main workflow evidence and retained receipt
E5 — independent reconstruction or reproduction
E6 — authorized external evaluation
E7 — peer-reviewed or institutionally accepted result
```

Current collaboration packet posture:

```text
Research framing: E1

Independent reconstruction: evidence asserted in broader program, not yet attached to this packet
Texas Tech/NVIDIA evaluation: E0 — none claimed
```

## Minimum evidence packet for an exploratory conversation

Before outreach, attach or link:

1. the research concept note;
2. this evidence map;
3. the minimal reproducible experiment specification;
4. one canonical positive case and at least three negative/drift cases;
5. machine-readable expected outcomes;
6. verifier instructions and environment requirements;
7. a result receipt containing input hashes, artifact versions, decision, and post-state hash;
8. explicit known limitations and non-claims;
9. verified repository paths and immutable commit references;
10. contact/program map sourced from current official Texas Tech records.

## Claims permitted by current evidence

Permitted:

> StegVerse has documented an experimental research framework and is preparing reproducible tests for authority reconstruction, transition admissibility, execution-boundary compliance, and evidence continuity in agentic AI systems.

Not permitted:

- Texas Tech or NVIDIA has evaluated or approved StegVerse.
- StegVerse has access to Texas Tech accelerated-computing infrastructure.
- Existing demonstrations establish general safety, security, alignment, or governance.
- A repository artifact constitutes independent validation.
- A successful local replay proves reconstructability across independent implementations.

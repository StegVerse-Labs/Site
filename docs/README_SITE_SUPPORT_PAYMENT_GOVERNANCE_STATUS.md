# Site Support Payment Governance Status Bundle

## Assumptions

This bundle is for `StegVerse-Labs/Site`.

The goal is to update the public support and StegFinCo pages so they reflect the validated support-payment governance chain.

## Done condition

The update is complete when:

```text
support.html shows support-payment governance status
stegfinco.html shows support-event certification status
both pages preserve non-entitlement language
both pages show StegPay → StegFinCo → Fin-Co-Lab → Fin-Co
both pages keep the Support StegVerse Research Stripe link
```

## Files to replace

```text
Site/support.html
Site/stegfinco.html
```

## Validated chain represented

```text
Site support button
→ StegPay evidence verification
→ StegFinCo support_event classification
→ Fin-Co-Lab support-event certification
→ Fin-Co compatibility validation
```

## Boundary rule

```text
Payment is evidence.
Governance creates authority.
StegFinCo controls effect.
Fin-Co constrains interpretation.
```

## Core relation

```text
t_i^- = expected_irreversibility - confidence_multiplier × uncertainty
L_total = sense + transmit + infer + authorize + actuate + stabilize
t_c = t_i^- - L_total
```

The formalism distinguishes physical irreversibility from practical irreversibility. A transition can remain physically reversible while the time required to sense, infer, authorize, actuate, and stabilize has already exceeded the remaining recovery interval.

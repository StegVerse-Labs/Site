# KnowledgeVault Conversation Continuity Paper Publication

## Public route

`papers/knowledgevault/conversation-continuity.html`

## Purpose

Publishes the bounded public HTML edition of the KnowledgeVault conversation-continuity position paper and exposes it through `Papers.html`.

## Source and implementation relationship

- Position-paper subject: Merkle-backed conversation continuity, retention classes, provenance-bearing reconstruction, and searchable durable consequences.
- Working prototype: `StegVerse-Labs/continuity-vault-kit`.
- Site role: public display and discovery surface only.

## Boundary

Publication on Site does not independently validate the paper's claims, grant execution authority, certify production readiness, or convert the working local-first prototype into a deployed conversational-memory service.

The direct PDF asset is not yet installed because the active connector path supports UTF-8 repository content but not binary file upload. The public HTML route is the stable LinkedIn-compatible publication URL. A direct PDF may be added later without changing the paper's canonical public route.

## Validation target

Site validation should confirm:

1. the paper route exists;
2. the `Papers.html` featured entry resolves to that route;
3. no workflow, transport, deployment, custody, execution, or authority claim changed;
4. the page describes the prototype as working and local-first while identifying the larger production architecture as incomplete.
